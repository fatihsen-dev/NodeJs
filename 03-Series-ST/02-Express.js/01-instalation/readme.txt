pacpage.json
- pacpage.json projeniz ile ilgili bilgileri tutar (version,isim,paketler,vs)
- npm init derseniz bütün özelikleri belirmeniz gerekicektir
- npm init -y derseniz otomatik bir adet oluşturur

node_modules
- node_modules paketlerinizin bulunduğu klasör
- node_modules klasörünü silerseniz npm i komutu ile pacpage.json içerisindeki (dependencies)objesi içerisindeki paketleri kurarsınız
